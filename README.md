## Full Subtractor using Half Subtractor
This repository contains the **verilog** **files** for designing a **full subtractor using half subtractors**. The repository contains both the design source and 
the corresponding **testbench** for both the half subtractor and full subtractor. The files are free to use for all. You can use these files as a reference in 
designing your own modules.
More information about Half Subtractor can be found [here] (https://www.tutorialspoint.com/digital-electronics/digital-electronics-half-subtractor.html) .
More intormation about Full Subtractor can be found [here] (https://www.tutorialspoint.com/digital-electronics/digital-electronics-full-subtractor.htm) .
